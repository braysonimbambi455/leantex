# Leantex Platform

An Integrated Service Booking, Marketing, and Customer Relationship Platform for Leantex Company Limited.

## Project Overview

Leantex is a web-based platform that centralizes service presentation, appointment scheduling, and customer records for Leantex Company Limited, a service-based SME offering:
- CCTV installation
- WiFi setup
- Accounting advice
- Computer sales
- Laptop repair

## Features

### Customer Features
- View services and pricing
- Book appointments online
- Receive email/SMS confirmations
- Make online payments
- View booking history
- Leave feedback

### Admin Features
- Manage services
- Manage bookings
- Manage customers
- Assign technicians
- View analytics
- Manage payments

### Technician Features
- View assigned jobs
- Update service status
- View schedule

## Technology Stack

- **Backend:** Python Django 4.2
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Database:** PostgreSQL
- **Payment Processing:** Stripe, PesaPal
- **Notifications:** Twilio (SMS), SendGrid (Email)
- **Version Control:** Git, GitHub
- **Hosting:** AWS/Azure/Render
